WellFit Magazine Cover

Wellfit Magazine is a personal brand magazine, which is created for Illustration class in the Multimedia Communication program from Yukon University. This project was designed with 3 different concepts (design mock-ups) for a magazine cover. This project was to photograph and design the layout of 3 different versions of a 'Magazine’. The covers will feature portrait photography demonstrating basic, 1 light, lighting techniques. You will design 3 different versions of a cover for a Magazine. Each cover will feature a different portrait headshot demonstrating a different lighting technique (using only 1 light). Portrait photos were original images. This assignment also showcase the understanding of basic retouching while demonstrating the skills using the type tools and preparing a document for printing.
