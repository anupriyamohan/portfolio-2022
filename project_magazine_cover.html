WellFit Magazine Cover

Wellfit Magazine is a personal brand magazine, which is created for Illustration class in the Multimedia Communication program from Yukon University.

This project was to photograph and design the layout of 3 different versions of a magazine cover. The covers feature portrait photography demonstrating basic lighting techniques. Each cover features a different portrait headshot. This assignment showcases basic photo retouching and typographic skills.
